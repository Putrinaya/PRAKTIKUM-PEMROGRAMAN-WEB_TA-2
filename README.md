Portofolio Pribadi-Putri Naiya Ramadhani
Proyek ini merupakan hasil dari tugas akhir mata kuliah Praktikum Pemrograman Web.
Website yang dibuat bertema portofolio pribadi, dengan tujuan untuk menampilkan profil, pengalaman, serta hasil karya dalam bentuk halaman web yang menarik dan mudah diakses.

Seluruh tampilan website dikembangkan menggunakan HTML dan CSS tanpa framework. Fokus utamanya adalah membangun struktur halaman yang rapi, menata elemen visual agar harmonis, serta menerapkan animasi ringan agar website terasa lebih interaktif.
